AutoEngine: Electrical Equipment Monitoring Dashboard / نظام اتمتة و مراقبة المعدات الكهربائية
هذا المشروع هو جزء من حلول AutoEngine المتخصصة في اتمتة الانظمة الكهربائيه. تم تطوير هذا الداشبورد لتقديم رؤية فورية وذكية لاداء المعدات , مما يسهل مراقبة البيانات وتحليلها بدقه.
/ This project is part of AutoEngie's specialized solutions for electrical system automation. 
This dashboard was develobed to provide real-time ,intelligent insights into equipment performance, facilitating precise data monitoring and analysis.

AutoEngine: Electrical Equipment Monitoring Dashboard / نظام اتمتة و مراقبة المعدات الكهربائيه.
 
 
 This project is part of AutoEngie's specialized solutions for electrical system automation.This dashboard was develobed to provide real-time ,intelligent insights into equipment performance, facilitating precise data monitoring and analysis. /هذا المشروع هو جزء من حلول شركة اوتو انجين المتخصصة في اتمتة الانظمة الكهربائية.
